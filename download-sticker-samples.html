Create your artwork!

Don't have professional software for artwork? Don't worry! Simply use any of the following websites or software.

Canva

Recommended!

Canva is an online program that allows you to create your own artwork from plenty of premade templates! This is best for beginners without prior Photoshop experience. While you can also use the app, however you will need to save the file as a PDF for print. Do note that this can only be done via the website (whether on mobile or desktop).

PhotoPea

Recommended!

PhotoPea is a free clone of Photoshop that opens right in your browser with nothing to download. This is best for those with prior experience with Photoshop. You can do some advanced image editing, create artwork from scratch, or use it to edit premade templates! Just save as a PSD and send that to us to print!

Google Docs/Slides

If you are on the go and need only basic artwork (eg plain text stickers), Google Docs and Google Slides will serve you well! Just type, insert any graphics you need, and then select "Share & Export" and then "Save As..." in order to export the file as a PDF for printing.

MS Office

If you have Microsoft Office, you can also use that to create your artwork! The online version is free to use without downloading anything. As with Google, simply insert your required text & graphics, and then Save As -> Download to PDF (or simply Save As PDF if you are using the desktop software!)
